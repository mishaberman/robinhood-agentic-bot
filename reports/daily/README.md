# Daily Reports

This folder is for local end-of-day Markdown writeups generated by the Codex monitor.

Expected file pattern:

```text
YYYY-MM-DD.md
```

For the low-friction local setup, leave the laptop awake with Codex open. The active Codex heartbeat monitor runs every 5 minutes during the market window and updates the daily report after the close.

If there are no actual monitor outputs for a day, the report should say that plainly instead of inventing alerts or trades.
